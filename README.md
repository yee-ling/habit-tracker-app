Habit Tracker App

Introduction
Habit Tracker App is a simple yet powerful productivity tool designed to help users build and maintain positive habits. The app allows users to create, manage, and track daily habits while gaining insights into their consistency over time.
Built with a focus on clean UI and usability, this app aims to make habit tracking effortless and motivating for everyday use.

Features
Create & Manage Habits
Users can add, edit, and delete habits based on their personal goals.
Daily Habit Tracking
Mark habits as completed for each day to build consistency.
Progress Visualization
Track performance and view habit completion trends over time.
Offline Data Persistence
All data is stored locally using Room Database, allowing the app to work without an internet connection.

Technologies Used
Kotlin – Main programming language
Android Studio – Development environment
Room Database – Local data persistence

Usage
Create new habits you want to track
Mark habits as complete daily
Monitor your progress and stay consistent
